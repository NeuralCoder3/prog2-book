The abstractions that higher programming languages (such as C or Java) provide are of course helpful because they save us work. They relieve the programmer from having to think about low-level details and enable portable code. Portable means that a program can be run on different machines that have possibly different instruction set architectures. One way of enabling portability is using a compiler.
