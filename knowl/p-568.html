In practice, one is interested in keeping \(\alpha\lt 1\) which means that, in the ideal case, no collision contains more than one key. To achieve this, on each insertion operation the load factor is checked and if it passes a certain threshold (0.75 is a common value) the hash table is resized. Note that when the hash table is resized, \(h_m\) also changes and all elements in have to be rehashed into the new table.
