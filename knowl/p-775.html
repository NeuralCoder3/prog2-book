Given a program \(\prgltr\) and a specification \((P,Q)\text{.}\) A state \(\sigma\) that satisfies the precondition is a test of \(\prgltr\text{.}\) A test is passed if \(\config\prgltr\sigma\terminates\config\CTerm{\sigma'}\) and \(\sigma'\models Q\text{.}\) If \(\sigma\) is a failure we say the test failed. A set of tests is called a test suite.
